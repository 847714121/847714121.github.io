CSS复合选择器

复合选择器就是将基础选择器组合在一起，主要包含
· 后代选择器
· 子选择器
· 并集选择器
· 伪类选择器

1.后代选择器（非常重要）

后代选择器又称包含选择器，可选择父元素里的子元素.其写法就是把外层标签写在前面，内层标签写在后面，中间用空格分隔，当标签发生嵌套时，内层标签就是外层标签的后代

元素1 元素2 { ... }

比如li即在ol里也可以在ul里，这样可以只选择其中的一个

  1. 我是ol的小il
  2. 我是ol的小il
  3. 我是ol的小il
  • 我是ul的小li
  • 我是ul的小li
  • 我是ul的小li

2.子选择器（重要）

子元素选择器（子选择器）只能选择作为某元素的最近一级子元素。简单理解就是选亲儿子元素。

用大于号隔开

元素1>元素2 {样式声明}
    我是儿子
  • 我是孙子

3.并集选择器（重要）

并集选择器可以选择多组标签，同时为他们定义相同的样式，通常用于集体声明

用，分割

标签2, 标签2 {样式声明}

4.伪类选择器

鼠标放到某个文字上，文字会发送改变都是通过伪类选择器来实现的，伪类选择器用于向某些选择器添加特殊的效果，比如给连接添加特殊效果，或选择第1个，第n个元素

伪类选择器最大的特点是用冒号（:）表示

伪类选择器有很多种，比如连接伪类，结构伪类等

1.连接伪类选择器

a:link          /*选择所有未被访问的连接*/
a:visited     /*选择所有已被访问的连接*/
a:hover      /*选择鼠标指针位于其上的连接*/
a:active     /*选择活动连接(鼠标按下未弹起的连接)*/

你把鼠标放上来试试，会变颜色哦！你在点击试试，也会变颜色哦！

连接伪类注意事项
1.为了确保生效，请按照LVHA的顺序声明：link-:visted-:hover-:active
2.因为a连接在浏览器中具有默认样式，所以我们在实际开发中需要给链接单独指定样式。

连接伪类选择器在实际工作开发中的写法：

/*a 是标签选择器 所有连接*/
a {
  color: gray;
}
/* :hover 是连接伪类选择器 鼠标经过 */
a:hover {
  color: red; /* 鼠标经过的时候，有原来的灰色变成红色 */
}

这样的效果就是连接平时是一个颜色，鼠标放上去以后变成另一个颜色

5.focus伪类选择器

:focus伪类选择器用于选择获得交点的表单元素。

焦点就是光标，一般情况下 input 类表单元素才能获取，因此这个选择器主要针对于表单元素来说。

input:focus {
  background-color: yellow;
}

鼠标点击地下的小框底色就会变哦！
