CSS复合选择器

复合选择器就是将基础选择器组合在一起，主要包含
· 后代选择器
· 子选择器
· 并集选择器
· 伪类选择器

1.后代选择器（非常重要）

后代选择器又称包含选择器，可选择父元素里的子元素.其写法就是把外层标签写在前面，内层标签写在后面，中间用空格分隔，当标签发生嵌套时，内层标签就是外层标签的后代

元素1 元素2 { ... }

比如li即在ol里也可以在ul里，这样可以只选择其中的一个

  1. 我是ol的小il
  2. 我是ol的小il
  3. 我是ol的小il
  • 我是ul的小li
  • 我是ul的小li
  • 我是ul的小li

2.子选择器（重要）

子元素选择器（子选择器）只能选择作为某元素的最近一级子元素。简单理解就是选亲儿子元素。

用大于号隔开

元素1>元素2 {样式声明}
    我是儿子
  • 我是孙子

3.并集选择器（重要）

并集选择器可以选择多组标签，同时为他们定义相同的样式，通常用于集体声明

用，分割

标签2, 标签2 {样式声明}

4.伪类选择器

鼠标放到某个文字上，文字会发送改变都是通过伪类选择器来实现的，伪类选择器用于向某些选择器添加特殊的效果，比如给连接添加特殊效果，或选择第1个，第n个元素

伪类选择器最大的特点是用冒号（:）表示

伪类选择器有很多种，比如连接伪类，结构伪类等

1.连接伪类选择器

a:link          /*选择所有未被访问的连接*/
a:visited     /*选择所有已被访问的连接*/
a:hover      /*选择鼠标指针位于其上的连接*/
a:active     /*选择活动连接(鼠标按下未弹起的连接)*/

你把鼠标放上来试试，会变颜色哦！你在点击试试，也会变颜色哦！

连接伪类注意事项
1.为了确保生效，请按照LVHA的顺序声明：link-:visted-:hover-:active
2.因为a连接在浏览器中具有默认样式，所以我们在实际开发中需要给链接单独指定样式。

连接伪类选择器在实际工作开发中的写法：

/*a 是标签选择器 所有连接*/
a {
  color: gray;
}
/* :hover 是连接伪类选择器 鼠标经过 */
a:hover {
  color: red; /* 鼠标经过的时候，有原来的灰色变成红色 */
}

这样的效果就是连接平时是一个颜色，鼠标放上去以后变成另一个颜色

5.focus伪类选择器

:focus伪类选择器用于选择获得交点的表单元素。

焦点就是光标，一般情况下 input 类表单元素才能获取，因此这个选择器主要针对于表单元素来说。

input:focus {
  background-color: yellow;
}

鼠标点击地下的小框底色就会变哦！

CSS的元素显示模式

了解元素的显示模式可以更好的让我们对网页进行布局

1什么是元素显示模式？

作用：网页额的标签非常多，在不同的地方会用到不同类型的标签，了解他们的特点可以更好的布局我们的网页。
元素显示模式就是元素（标签）以什么方式进行显示，比如div自己占一行，span一行可以放多个。
HTML元素一般可以分为块元素和行内元素两种类型

1.块元素

常见的块元素有h1-h6,p,div,ul,ol,li等，其中div是典型的块元素
块元素额特点：
① 自己独占一行
② 高度，宽度，外边距和内边距都可以控制。
③ 宽度默认是容器（父级宽度）的100%
④ 是一个容器及盒子，里面可以放行内或块级元素。

注意：
· 文字类元素里面不能再放块级元素
· p里面不能放div

2.行内元素

常见的有a，strong，b，em，i，del，span等，其中span是最典型的行内元素，有的地方也将行内元素称为内联元素
行内元素的特点：
① 相邻行内元素在一行上，一行可以显示多个。
② 高、宽的设置是无效的。 ③ 默认宽度就是它的本身内容的宽度。
④ 行内元素只能容纳文本或其他行内元素

注意
· 连接里不能再放连接
· 特殊情况连接a 里面可以放块级元素，但是给a转换一下块级模式最安全。

3.行内块元素

在行内元素中有几个特殊的标签--img /、inout /、td，它们同时具有块元素和行内元素的特点
行内块元素的特点:
① 和相邻行内元素在一行上，但它们之间会有空白间隙。一行可以显示多个
② 默认宽度就是它本身内容的宽度。
③ 高度，行高，外边距，内边距都可以控制

4.显示模式的转换

特殊情况下，我们需要元素模式的转换，简单理解：一个模式的元素需要另一个模式的特性，比如想要增加a的触发范围

· 转换为块元素： display:block

· 转换为行内元素： display:inline

· 转换为行内块元素： display:inline-block

CSS的背景

通过背景属性，可以给页面元素添加背景样式
背景属性可以设计背景颜色，背景图片，背景平铺，背景图片位置、背景图片固定等。

1.背景颜色

background-color，可以设置为透明:transparent（默认的）\color值

2.背景图片

background-image属性描述了元素的背景图像，实际开发中常见于logo或者一些装饰性小图片或者是超大的背景图片，优点是非常便于通知位置（精灵图也是一种运用场景）

background-image : none | url (url)

3.背景平铺

如果需要爱HTML页面上对背景图像进行平铺，可以使用background-repeat属性

background-repeat : repeat | no-repeat | repeat-x repeat-y

4.背景图片位置（非常重要）

利用background-position可以修改图片在背景中的位置

background-position: x y;

可以使用方位名词或精确单位。如果缺失一个参数值，则默认居中

参数值 说明
length 百分数|由浮点数字和单位标识组成的长度值
position top|center|bottom|right等方位名词

5.背景图像固定（背景附着）

通过background-attachment属性设置是否固定或者随着页面的其余部分滚动，可以用来做视差滚动

background-attachment : scroll | fixed

6.背景复合写法

为了简化背景属性代码，我们可以将这些属性合并简写在同一个属性background中。没有特定的顺序要求，一般写为：
background:背景颜色 背景图片地址 背景平铺 背景图像滚动 背景图片位置

7.背景色半透明

不用background-color，用background时，颜色部分用rgba(0, 0, 0, 0.3)，最后一个参数是透明度
