1.精灵图

1.1 为什么需要精灵图

一个网页中往往会用到很多小的背景图像作为修饰，当网页中的图像过多时，服务器就会频繁的接收和发送图片的请求，造成服务器压力过大，这将大大降低页面的加载速度。

因此，为了有效地减少服务器接受和发送请求的次数，提高页面的加载速度，出现了CSS精灵图技术

核心原理：将网页中的一些小背景图片整合到一张大图中，这样服务器只需要一次请求就可以了。

精灵图（sprites）的使用

使用精灵图核心：
1. 精灵技术主要针对背景图片使用。就是把小背景图片整合到一张大图中。
2. 这个大图片也称为sprites精灵图或者雪碧图
3. 移动背景图片位置，此时可以使用background-position
4. 移动的距离就是这个目标图片的x和y坐标，注意网页中的坐标是有所不同的。
5. 一般都是往上和往右移，所以都是负值。
6. 使用精灵图的时候需要精准测量，每个小背景图片的大小和位置

2. 字体图标

2.1 字体图标的产生

字体图标使用场景：主要用于显示网页中通用、常用的一些小图标。

精灵图有诸多优点，但缺点也很明显。
1. 图片文件还是比较大
2. 图片本身放大和缩小会失真。
3. 一旦图片制作完成想要更换非常复杂
此时，有一种技术的出现很好的解决了以上问题，就是字体图标iconfont
字体图标展示的是图标，本质是字体

优点：
1.轻量级
2. 灵活性：本质是文字，可以随意更改样式
3.兼容性：几乎所有浏览器都支持

2.2 字体图标的下载

字体图标是一些网页常见的小图标，我们直接从网页上下载即可。因此可以分为这几步：
1. 字体图标的下载
2. 字体图标的引入（引入到html页面中）
3. 字体图标的追加（以后添加新的小图标）

推荐下载网址：

icomoon字库 icomoon.io

成立较早，内容繁多，非常全面，唯一的遗憾是国外服务器，打开网速较慢

阿里iconfont字库 www.iconfont.cn

2.3 字体图标的引入

下载完毕后，注意原先的文件不要删

1.把下载包里的fonts文件夹放在根目录下
2. 在CSS样式中全局声明字体：简单理解把这些字体文件通过CSS引入到我们的页面当中。
一定注意字体文件路径的问题。

@font-face {
  font-family: 'icomoon';
  src: url('fonts/icomoon.eot?7kkyc2');
  src: url('fonts/icomoon.eot?7kkyc2#iefix') format('embedded-opentype'),
    url('fonts/icomoon.ttf?7kkyc2') format('truetype'),
    url('fonts/icomoon.woff?7kkyc2') format('woff'),
    url('fonts/icomoon.svg?7kkyc2#icomoon') format('svg');
  font-weight: normal;
  font-style: normal;
}

2.4 字体图标的追加

如果原先的图标不够用，我们需要添加新的字体图标到原来的字体中。
只需把压缩包里的selection.json重新上传，然后选中自己想要更新的图标，重新下载压缩包，并替原来的文件即可

5 vertical-align属性应用

CSS的vertical-align属性应用场景：经常用于设置图片或者表单（行内块元素）和文字垂直对齐。只针对行内块或行内元素有效。

语法：

vertical-align : baseline | top | middle | bottom

baseline是基线对齐，就是文字底下所在的线。默认是基线对齐

5.2 盒子底部默认空隙问题

因为图片默认是基线对齐，所以会给文字留空隙，可以改变对齐方式，只要不是基线对齐就可以去掉。
把图片转换为块级元素也可以，因为块级元素没有vertical-align属性

6. 溢出的文字省略号显示

6.1 单行文本溢出省略号显示

必须满足三个条件

1. 必须强制一行内显示文本
white-space: nowrap; （默认normal 自动换行）
2. 超出部分隐藏
overflow: hidden;
3. 文字用省略号代替超出部分
text-overflow: ellipsis

6.2 多行文本溢出省略号显示

有较大的兼容性问题，适用于webkit浏览器或移动端

代码：

overflow: hidden;
text-overflow: ellipsis;
/* 弹性伸缩盒子模型显示 */
display: -webkit-box;
/* 限制在一个块元素显示的文本行数 */
-webkit-line-clamp: 2;
/* 设置或检索伸缩对象的子元素的排列方式 */
-webkit-box-orient: vertical;

常见的布局技巧

1. margin负值的应用

多个盒子放一起时，边框会1+1等于2，变粗，将margin设为-1可以解决这个问题，让边框重叠，只显示一个

进阶：想让鼠标经过时边框变色，因为上面说的右边盒子会压住左边盒子，所以右边框直接写不会变色，只需将所选中的盒子变为相对定位relative，即可。或者添加Z轴(z-index)

2. 文字浮动效果使用小妙招

当一个盒子是一边为图片，一边为文字，只要运用浮动（图片）不占位置，但文字（标准流中）会环绕来做

3. 行内块巧妙布局

一个盒子添加text-align: center，里面的行内元素和行内块元素都可以居中对齐，可以做翻页标签
