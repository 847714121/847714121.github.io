1.2 HTML5新增多媒体标签

新增的多媒体标签主要包含两个：

1. 音频： audio
2. 视频： video

1.视频 video

当前video元素支持三种视频格式(mp4，webM，Ogg)：尽量使用mp4格式（IE、safari只支持mp4）

语法：

video src = "文件地址" controls = "controls"

常见属性

autoplay : autoplay 视频就绪自动播放
controls : controls 向用户显示播放控件
width : xx px 播放器宽度
height : xx px 播放器高度
loop : loop 循环播放
preload : auto/none 规定是否预加载视频
src : url 视频地址
poster : imgurl 加载等待的画面图片
muted : muted 静音播放

2.音频 audio

当前audio支持三种音频格式：MP3(都支持)，Wav、Ogg

语法

audio src = "文件地址" controls = "controls"

2. CSS3新特性

2.1 属性选择器

利用属性选择器可以通过属性来选择，如
input[value]
input[value = xxxx]
input[value ^= xxx] 以xxx开头的元素
input[value $= xxx] 以xxx结尾的元素
input[value *= xxx] 中间有xxx就会被选

类选择器、伪类选择器、属性选择器权重都是10

2.2 结构伪类选择器

E :first-child 选择元素E里的第一个孩子
last-child
nth-child(n) n可以是数字，也可以是关键字(odd、even)
E :first-of-type
last-of-type
nth-of-type(n)

2.4 伪元素选择器(重点)

伪元素选择器可以帮助我们利用CSS创建新标签，而不需要HTML标签，从而简化HTML结构

element::before{ 样式 } 在元素前面插入内容
element::after{ 样式 } 在元素后面插入内容

注意：

· before和after创建一个元素，但是输入行内元素
· 新创建的元素在文档树中找不到，所以称为伪元素
· before 和 after 必须有content属性
· 伪元素选择器和标签选择器一样，权重为1

2.5 CSS3盒子模型

CSS3中可以通过 box-sizing 来指定盒模型，有2个值：即可指定为content-box(默认)、border-box(不在撑大盒子)
